Om Bröllopet

Efter många år som förlovade, flera barn och mycket roligt så bestämde vi oss för ett datum för giftermålet!

När vi väl hittade stället vi ville ha festeten på så föll alla bitarna på plats. Mer information hittar du längre ner på sidan.

Vi ser verkligen fram mot denna tillställning och hoppas att vi kommer ha en fantastisk dag tillsammans!

Vigselceremoni

För att göra det smidigt för alla så valde vi en kyrka nära Växjö och nära till festlokalen.

Vigseln startar klockan 16:00

Festen

Det var inte fören vi hittade Rinkabygård som vi verkligen bestämde oss för att sätta ett datum för vigseln.

Lokalen har allt som vi tyckte var viktigt för att få till en riktigt bra bröllopsfest.

vårt fokus har varit att det skall bli så roligt som möjligt för alla inblandade och inte så mycket fokus på seder och tradition.

Vi vill därför att du gör ditt yttersta för att verkligen det så trevligt som möjligt!

RSVP

För att vi skall kunna få till allt det praktiska så behöver vi veta om du/ni kommer.

Fyll i nedanstånde formulär och berätta om ni kan komma eller inte, vi behöver få in svar innan 2017-02-01.

T.ex. Allergier eller annat övrigt som kan vara bra för oss att veta.
